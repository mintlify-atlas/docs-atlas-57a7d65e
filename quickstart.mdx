Get up and running with Worlds Platform in just a few minutes. This guide will walk you through installing the SDK, setting up your first world, and running your first SPARQL query.

## Choose Your Setup

Worlds Platform can run in two modes:

<CardGroup cols={2}>
  <Card title="Local Server" icon="laptop">
    Run the Worlds Platform server locally on your machine. Perfect for development and testing.
  </Card>
  <Card title="Cloud" icon="cloud">
    Connect to a hosted Worlds Platform instance. Production-ready and scalable.
  </Card>
</CardGroup>

## Installation

<Steps>
  <Step title="Install the SDK">
    Choose your preferred package manager:

    <CodeGroup>
      ```bash Deno
      deno add jsr:@wazoo/worlds-sdk
      ```

      ```bash npm
      npx jsr add @wazoo/worlds-sdk
      ```

      ```bash pnpm
      pnpm dlx jsr add @wazoo/worlds-sdk
      ```

      ```bash Yarn
      yarn dlx jsr add @wazoo/worlds-sdk
      ```
    </CodeGroup>
  </Step>

  <Step title="Set up a local server (Optional)">
    If you want to run Worlds Platform locally, install the server:

    ```bash
    deno install -A --name worlds-server jsr:@wazoo/worlds-server
    ```

    Create a `.env` file with required configuration:

    ```bash .env
    LIBSQL_URL=file:./worlds.db
    GOOGLE_API_KEY=your-google-api-key
    WORLDS_API_KEY=your-api-key
    ```

    Start the server:

    ```bash
    deno task start
    ```

    The server will start on `http://localhost:8000` by default.

    <Note>
      For cloud usage, skip this step and use the hosted API endpoint at `https://api.wazoo.dev`.
    </Note>
  </Step>
</Steps>

## Your First World

Let's create a knowledge base (called a "world"), add some knowledge, and query it.

<Steps>
  <Step title="Initialize the SDK">
    Create a new file and initialize the Worlds SDK client:

    <CodeGroup>
      ```typescript Local Server
      import { WorldsSdk } from "@wazoo/worlds-sdk";

      const sdk = new WorldsSdk({
        baseUrl: "http://localhost:8000",
        apiKey: "your-api-key",
      });
      ```

      ```typescript Cloud
      import { WorldsSdk } from "@wazoo/worlds-sdk";

      const sdk = new WorldsSdk({
        baseUrl: "https://api.wazoo.dev",
        apiKey: "your-api-key",
      });
      ```
    </CodeGroup>
  </Step>

  <Step title="Create a world">
    Create your first knowledge base:

    ```typescript
    const world = await sdk.worlds.create({
      slug: "my-first-world",
      label: "My First World",
      description: "A knowledge base for learning Worlds Platform",
    });

    console.log("Created world:", world.id);
    ```

    <Tip>
      The `slug` is a URL-friendly identifier, while `label` is a human-readable name.
    </Tip>
  </Step>

  <Step title="Add knowledge (triples)">
    Add some knowledge to your world using SPARQL INSERT:

    ```typescript
    const worldId = world.id;

    // Add triples about people and their relationships
    await sdk.worlds.sparql(
      worldId,
      `
      INSERT DATA {
        <http://example.com/ethan> <http://schema.org/relatedTo> <http://example.com/gregory> .
        <http://example.com/gregory> <http://schema.org/givenName> "Gregory" .
        <http://example.com/gregory> <http://schema.org/jobTitle> "Engineer" .
        <http://example.com/ethan> <http://schema.org/givenName> "Ethan" .
      }
      `
    );

    console.log("Knowledge added successfully!");
    ```

    <Note>
      Knowledge in Worlds Platform is stored as RDF triples: subject-predicate-object statements.
      Each triple represents a fact about your data.
    </Note>
  </Step>

  <Step title="Query with SPARQL">
    Now let's query the knowledge we just added:

    ```typescript
    // Find all people related to Ethan
    const result = await sdk.worlds.sparql(
      worldId,
      `
      SELECT ?name ?title WHERE {
        <http://example.com/ethan> <http://schema.org/relatedTo> ?person .
        ?person <http://schema.org/givenName> ?name .
        OPTIONAL { ?person <http://schema.org/jobTitle> ?title . }
      }
      `
    );

    console.log("Query results:", JSON.stringify(result, null, 2));
    ```

    The result will look like:

    ```json
    {
      "head": {
        "vars": ["name", "title"]
      },
      "results": {
        "bindings": [
          {
            "name": {
              "type": "literal",
              "value": "Gregory"
            },
            "title": {
              "type": "literal",
              "value": "Engineer"
            }
          }
        ]
      }
    }
    ```
  </Step>
</Steps>

## Complete Example

Here's a complete working example that puts it all together:

```typescript example.ts
import { WorldsSdk } from "@wazoo/worlds-sdk";

// Initialize the client
const sdk = new WorldsSdk({
  baseUrl: "http://localhost:8000",
  apiKey: "your-api-key",
});

async function main() {
  // Create a world
  const world = await sdk.worlds.create({
    slug: "my-knowledge-base",
    label: "My Knowledge Base",
  });

  console.log("Created world:", world.id);

  // Add some knowledge
  await sdk.worlds.sparql(
    world.id,
    `
    INSERT DATA {
      <http://example.com/ethan> <http://schema.org/relatedTo> <http://example.com/gregory> .
      <http://example.com/gregory> <http://schema.org/givenName> "Gregory" .
      <http://example.com/ethan> <http://schema.org/givenName> "Ethan" .
    }
    `
  );

  // Query the knowledge
  const result = await sdk.worlds.sparql(
    world.id,
    `
    SELECT ?name WHERE {
      <http://example.com/ethan> <http://schema.org/relatedTo> ?person .
      ?person <http://schema.org/givenName> ?name .
    }
    `
  );

  console.log("Query results:", result);
}

main();
```

Run it with:

```bash
deno run -A example.ts
```
