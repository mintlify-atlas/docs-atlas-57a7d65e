The Worlds SDK is a TypeScript SDK for interacting with the Worlds API. It provides a simple and type-safe interface for managing knowledge graphs.

## Package Information

- **Package Name**: `@wazoo/worlds-sdk`
- **Version**: `0.0.1`
- **Entry Point**: `./mod.ts`

## Installation

Install the SDK using your preferred package manager:

<CodeGroup>

```bash npm
npm install @wazoo/worlds-sdk
```

```bash pnpm
pnpm add @wazoo/worlds-sdk
```

```bash yarn
yarn add @wazoo/worlds-sdk
```

```bash deno
# Add to your import map or import directly:
import { WorldsSdk } from "jsr:@wazoo/worlds-sdk";
```

</CodeGroup>

## Quick Start

Once installed, you can initialize the SDK with your API credentials:

```typescript
import { WorldsSdk } from "@wazoo/worlds-sdk";

const sdk = new WorldsSdk({
  baseUrl: "https://api.worlds.example.com",
  apiKey: "your-api-key-here"
});

// List all worlds
const worlds = await sdk.worlds.list();
console.log(worlds);
```

## TypeScript Support

The SDK is written in TypeScript and provides full type definitions out of the box. No additional type packages are required.
