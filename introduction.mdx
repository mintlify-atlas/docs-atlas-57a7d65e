## What is Worlds Platform?

Worlds Platform is the open-source **auto-memory** and context layer for AI. It's a REST API designed to manage, query, update, and reason over [SPARQL](https://www.w3.org/TR/sparql11-overview/)-compatible knowledge bases at the edge.

Worlds Platform provides an interoperable context graph to improve agent reasoning, moving beyond similarity matches to true semantic understanding.

## Why Worlds?

<Note>
**World memory is not just RAG.** Worlds Platform focuses on symbolic memory. It understands relationships, hierarchies, and logic, moving beyond similarity matches to true reasoning.
</Note>

Traditional RAG systems rely on similarity search, which can miss important context. Worlds Platform uses SPARQL and RDF to create a structured knowledge graph that AI agents can reason over, enabling:

- **Relationship understanding**: Know not just what entities exist, but how they relate to each other
- **Logical inference**: Derive new knowledge from existing facts
- **Temporal context**: Track how knowledge evolves over time
- **Cross-domain reasoning**: Connect information across different knowledge domains
